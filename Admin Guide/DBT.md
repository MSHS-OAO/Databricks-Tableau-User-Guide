# DBT

[Back to Table of Contents](../README.md#table-of-contents)

DBT (Data Build Tool) will be the framework we use for all transformations in our ETL process. DBT will be used as a scheduled task in a projects ETL job. It will be the last step of all ETLs after all data has been moved to the databricks catalog via notebooks.

<u>**Note**</u>: This guideline will be very basic and directed at the common workflows our team will use. For a more in depth review of what DBT is capable of with live examples please refer to this youtube playlist:
[DBT Youtube Tutorial](https://youtube.com/playlist?list=PLy4OcwImJzBLJzLYxpxaPUmCWp8j1esvT&si=c0_Iv0svNnkAebLZ)
